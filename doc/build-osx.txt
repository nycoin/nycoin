-----------------------------------------------------------------------------------------------------
Mavericks build instructions
-----------------------------------------------------------------------------------------------------

The latest version of the newyorkc-qt.pro file is geared more towards building on Mavericks using
brew for dependencies.

1) Install Homebrew - http://brew.sh/

2) Make sure you have the following brewed:
brew install boost
brew install berkeley-db
brew install libtool
brew install miniupnpc
brew install openssl
brew install qt

3) Make sure you have the Xcode developer command line tools installed (brew will instruct you to install them as well)

4) You can now try to build the QT file. From the newyorkc folder in a command prompt, try:
qmake "USE_UPNP=1" newyorkc-qt.pro


5) You can now try to build: 
make -f Makefile
